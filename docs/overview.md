# ChainMove Overview

## Summary

ChainMove is a mobility payments project being repositioned for the Stellar ecosystem. Its core thesis is simple: transport is a daily, high-frequency payment use case, and Stellar is well suited to settle those payments cheaply and transparently.

## Product Direction

The product combines a rider and driver application layer with a Stellar settlement layer. Riders pre-fund trips, drivers receive programmable payouts, and operators gain verifiable records for fare collection and disbursement.

## Target Users

- riders who need predictable, auditable trip payments
- drivers who need faster payout and reliable earnings records
- fleet operators and transport cooperatives that need settlement rails
- institutions that want to fund or issue transport credits

## Primary Use Cases

- on-demand ride escrow and payout
- local transport fare settlement with stable assets
- fleet and cooperative driver disbursements
- closed-loop transport credits for employers, campuses, and communities

## Why It Fits Stellar

Stellar is a payments-first blockchain with strong asset support. That makes it a practical fit for:

- low-value but frequent consumer payments
- stablecoin settlement for drivers and operators
- issued transport credits
- corridor-specific local asset models supported by anchors

## Honest Repository State

The existing repository still contains a previous Internet Computer prototype. That code demonstrates the transport product direction but not the final chain choice. The current submission packages the repo for a Stellar-native next phase rather than claiming the migration is already complete.
